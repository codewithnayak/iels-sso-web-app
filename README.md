# iels-sso-web-app

A modern Single Sign-On (SSO) web application built using **React + Vite**, designed for integration with the **IELS Police Station Portal**.  
This project provides the frontend UI for authentication, login flows, and SSO redirection into the main portal.

---

## 🚀 Tech Stack

- **React 18**
- **Vite**
- **TypeScript**
- **Bootstrap / Custom UI**
- **Axios**
- **React Router**
- **Docker**
- **Helm (for GKE deployment)**
